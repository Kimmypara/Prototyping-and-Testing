# Feature: Reset Password 
## What should it do?
Create a password for first time login. 

## Why does it matter?
Allow new users access their account. 

## Inputs
- e-mail 
- password 
- confirm password 
- reset password 

## Outputs
- password is saved in DB
- user is redirected to the login page
