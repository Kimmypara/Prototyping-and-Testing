# Feature: Notification System 
## What should it do?
Highlights any new activity example brief or assignments 

## Why does it matter?
Helps users to find any new activity. 

## Inputs
- Teacher uploads a new file / brief
- Teacher creates a new assignment
- Teacher adds or updates grades

## Outputs
- Notification appears on My Units (e.g. red circle “1”)
- Notification shown inside specific unit (e.g. grading section)

## Key components
- Form
- Button
- Database
