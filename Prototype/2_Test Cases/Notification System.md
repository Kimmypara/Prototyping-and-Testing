# Test plan: Notification System 

## Persona
People who have new activity on My Units 

## Scenario
The user logs into the platform and wants to quickly notice any new activity in **My Units**.

## Tasks
1. Check for new activities 
2. Identify which unit has a new update.
3. Open the unit and find the new grade, assignment, or file.

## Success Metric
4/5 users identify any new activity quickly 

## Expected Outcome
Notification disappears after viewing it.

## Possible Failure Points
- The badge disappears too early before the user sees the new activity.
